# Architecture Decision Records

Architecture Decision Records capture significant, durable technical decisions that refine RFCs without contradicting them.

## Statuses

- Proposed
- Accepted
- Superseded
- Rejected
- Deprecated

## Naming

Use `ADR-NNN-short-title.md`, starting at `ADR-001`.

## Required sections

- Context
- Decision
- Alternatives considered
- Consequences
- Risks
- Rollback or migration
- References

Use [ADR-TEMPLATE.md](ADR-TEMPLATE.md) for every new decision.
