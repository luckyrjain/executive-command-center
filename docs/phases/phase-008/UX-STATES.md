# Phase 8 UX States

Surfaces: workspace switcher, members/invitations, sharing review, delegation inbox, shared activity and removal/transfer. Always show current workspace, visibility and accountable owner.

Required states: invitation expired, membership pending/suspended, permission changed, access revoked, delegation awaiting acceptance, ownership conflict and member removal blocked. Sharing previews exactly what becomes visible. Destructive removal and broad grants require deliberate confirmation. WCAG 2.2 AA applies.
