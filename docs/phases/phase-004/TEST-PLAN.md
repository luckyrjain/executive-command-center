# Phase 4 Test Plan

Cover provider adapters, deterministic routing, eligibility, budgets, timeout/cancel, circuit breakers, structured validation, tool allowlists, retries and fallback. Run versioned task evaluations and compare policy candidates. Adversarial fixtures include prompt injection, data exfiltration, malicious tool output, schema confusion and oversized context.

Verify redacted traces, no secret leakage, workspace isolation, remote-provider opt-in, AI-disabled behavior, backup/restore and reproducibility. Benchmark routing, local/remote latency and cost. Browser acceptance exercises generation, evidence, correction, cancellation and degraded states.
