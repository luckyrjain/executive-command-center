# Phase 4 UX States

AI-enhanced surfaces distinguish deterministic content from generated content and show source, freshness and confidence. Required states: AI disabled, local model unavailable, remote use not permitted, budget exceeded, timed out, cancelled, invalid output, degraded fallback and stale result. Failure never blocks the deterministic core flow.

Users can inspect evidence, retry within policy, correct feedback and discard output. No anthropomorphic certainty, hidden background activity or deceptive progress. Core flows meet WCAG 2.2 AA.
