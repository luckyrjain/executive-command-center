# Phase 5 UX States

Primary surfaces: workflow builder, simulation, policy review, approval inbox and run history. Clearly distinguish proposed, approved, active, paused and revoked. Show next scheduled run, authority scope and side effects.

Required states include waiting approval, expired approval, policy revoked, paused, retrying, unknown external outcome, partially compensated and kill switch active. Destructive/high-impact approvals require deliberate confirmation. All flows are keyboard accessible and meet WCAG 2.2 AA.
