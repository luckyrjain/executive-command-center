# Phase 9 UX States

Admin surfaces: identity, directory sync, policy, keys/residency, retention/legal hold, audit export, controls, quotas and break-glass review. Required states include SSO validation failure, SCIM partial sync, policy conflict, key rotation, legal hold conflict, export generation/failure, quota warning/block and emergency access active.

High-impact changes show affected scope, simulation and rollback. End users can see applicable policies and privacy implications. Administrative access never implies private-content visibility. WCAG 2.2 AA applies.
